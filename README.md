# cybersecurity_gamee

# Küberturvalisuse Põgenemistuba (Cybersecurity Escape Room)

## Mängu kirjeldus / Game Description

See on interaktiivne küberturvalisuse õppemäng eesti keeles, mis on loodud 7.-9. klassi õpilastele.
Mäng on täiesti toimiv ühe HTML-failina ja ei vaja serverit - avage lihtsalt `index.html` faili veebilehitsejas.

This is an interactive cybersecurity educational game in Estonian language, created for 7th-9th grade students.
The game is fully functional as a single HTML file and requires no server - just open `index.html` in a web browser.

## Kuidas mängida / How to Play

1. Avage `index.html` faili oma veebilehitsejas / Open `index.html` in your web browser
2. Klõpsake "Alusta seiklust" / Click "Start Adventure"
3. Läbige neli küberturvalisuse väljakutset / Complete four cybersecurity challenges
4. Valige õiged vastused klõpsates / Select correct answers by clicking
5. Jõudke võiduekraanile! / Reach the victory screen!

## Tubade kirjeldus / Room Descriptions

### Tuba 1: Tugeva parooli saladus / Room 1: Strong Password Secret
Õpi, mis teeb paroolist tugeva ja turvalise.
Learn what makes a password strong and secure.

### Tuba 2: Turvalised ühendused / Room 2: Secure Connections
Tuvasta turvalised võrguelemendid ja praktikad.
Identify secure network elements and practices.

### Tuba 3: Caesari šiffer / Room 3: Caesar Cipher
Dekrüpteeri krüpteeritud sõnum kasutades õiget nihkeväärtust.
Decrypt an encrypted message using the correct shift value.

### Tuba 4: Õngitsusrünnaku tuvastamine / Room 4: Phishing Detection
Tuvasta kahtlased elemendid võlts e-kirjas.
Identify suspicious elements in a fake email.

## Õpiväljundid / Learning Outcomes

Pärast mängu läbimist mõistavad õpilased:
After completing the game, students will understand:

1. Kuidas luua tugevaid paroole / How to create strong passwords
2. Kuidas kasutada turvaliselt võrke / How to use networks securely
3. Mis on krüpteerimine ja kuidas see töötab / What encryption is and how it works
4. Kuidas ära tunda õngitsusrünnakuid / How to recognize phishing attacks
